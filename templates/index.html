Tutorial

Cara Pakai 🚀

  • 1

    Install ekstensi Cookie-Editor di Chrome/Firefox.

  • 2

    Login ke Instagram, buka ekstensi, klik Export → Export as JSON untuk menyalin cookies.

  • 3

    Isi semua form di samping, termasuk menempelkan (paste) cookies yang tadi disalin.

  • 4

    Klik Start Scraping dan tunggu proses selesai. Tidak perlu login manual lagi.
